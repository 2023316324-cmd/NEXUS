# NEXUS IT Solutions - Website

Site institucional da Nexus IT Solutions, empresa especializada em soluções de TI, cibersegurança e infraestrutura em nuvem.

## Como Abrir o Visual (How to Open the Website)

### Opção 1: Abrir diretamente no navegador
1. Localize o arquivo `index.html` na pasta do projeto
2. Clique duas vezes no arquivo ou clique com botão direito e selecione "Abrir com" → seu navegador preferido (Chrome, Firefox, Edge, etc.)

### Opção 2: Usando Live Server (Recomendado para desenvolvimento)
1. Instale a extensão "Live Server" no Visual Studio Code
2. Abra a pasta do projeto no VS Code
3. Clique com botão direito no arquivo `index.html`
4. Selecione "Open with Live Server"

### Opção 3: Usando Python (servidor local)
```bash
# Python 3
python -m http.server 8000

# Depois abra no navegador: http://localhost:8000
```

### Opção 4: Usando Node.js
```bash
# Instale o http-server globalmente
npm install -g http-server

# Execute na pasta do projeto
http-server

# Abra no navegador o endereço indicado (geralmente http://localhost:8080)
```
